üõçÔ∏è Daily Operations Guide

  1. Opening a Session:
    • Go to Point of Sale > Dashboard
    • Click "New Session"
    • Enter opening balance (optional)
  2. Processing Sales:
    • Add products to the cart
    • Click "Pay" to process payment
    • Fiscal receipt will automatically print
    • Digital copy saved in FiscalNet format
  3. Cash Management:
    • Use "Cash In" for adding money to the register
    • Use "Cash Out" for removing money from the register
    • All transactions are automatically logged
  4. Closing Procedures:
    • Print Z-Report before closing session
    • Verify daily totals
    • Close session
